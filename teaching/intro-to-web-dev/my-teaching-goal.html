Hello Designers to Be! 

I am your instructor for this class, Juliet Crocco. My goal for this course is to help you learn enough about web development and HTML and CSS that you feel comfortable saying that you can hand-code a website using HTML and CSS and you are ready for learning JavaScript. If you do not need to take JavaScript, then I hope that you feel comfortable telling future employers that you can edit their websites that use HTML and CSS and you can read HTML and CSS code enough to understand what it is doing as a visual on a webpage.

This course has been redesigned from using a textbook to this version which is called an OER (Open Educational Resource) course. That means that we are using materials from different developers and resources all over the web that allow for sharing of content. 

The web is a very open environment. Since its early beginnings the web has been about openly sharing information. Therefore, learning how to use it and create in it is also just as open. Textbooks are awesome, don't get me wrong. They are super comprehensive and ideally will have everything you need in one place, however, in today's society, the Internet and the technology behind it are still changing all the time. Textbooks cannot keep up with those changes, so relying on industry standard information from current web developers like the stuff found on MDN (Mozilla Developer Network) Web Docs, is really helpful.

My hope is that as much as you learn skills in coding in this class, you will also learn how to find what you need online. You will need to be able to figure stuff out on your own. So, before asking for my help, do a Bing search (yes, I did just say Bing). 

Bing now has Chat GPT integrated into it so that you can get really comprehensive help and ask follow up questions right in the search engine.
